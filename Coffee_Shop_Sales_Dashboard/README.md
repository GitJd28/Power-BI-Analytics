# Coffee Shop Sales Dashboard

## Project Overview

This project analyzes coffee shop sales data using Power BI. The goal was to identify top-performing products, store performance, sales trends, and customer purchasing patterns to support business decision-making.

## Dataset

The dataset contains transaction-level sales data from multiple coffee shop locations and includes:

* Transaction Date & Time
* Product Category
* Product Type
* Product Details
* Store Location
* Quantity Sold
* Unit Price

## Tools Used

* Power BI Desktop
* Power Query
* DAX
* Microsoft Excel Dataset

## Dashboard Features

### Sales Performance

* Total Revenue Analysis
* Revenue Trend Over Time
* Revenue by Store Location

### Product Analysis

* Revenue by Product Category
* Top Selling Products
* Product Performance Comparison

### Store Analysis

* Store-wise Revenue Comparison
* Best Performing Store Identification

### Time Analysis

* Monthly Revenue Trends
* Peak Sales Period Analysis

## Key Insights

* Coffee products generated the highest revenue among all categories.
* Tea products were the second-largest contributor to sales.
* Revenue varied across store locations, highlighting differences in store performance.
* Certain product types consistently outperformed others in sales revenue.
