# Sales Dashboard Project

## Overview

This project is part of my Power BI learning journey. It demonstrates the creation of an interactive sales dashboard using Power BI to analyze sales performance and key business metrics.

## Files

* `sales_report_jd.pbix` – Power BI dashboard file
* `sales_data.csv` – Dataset used for analysis
* `Images/` – Dashboard screenshots

## Key Insights

* Sales performance analysis
* Revenue trends visualization . 
* Product and regional performance tracking
* Interactive filtering and drill-down capabilities.

## Tools Used

* Power BI
* DAX
* Power Query
* CSV Data Sources

## Learning Outcomes

Through this project, I practiced:
* Data cleaning and transformation
* Data modeling
* Creating interactive visualizations
* Building KPI cards and reports
* Using DAX for calculations
